# eArc-cast

earc/cast does help you to cast data represented by objects/array into or onto 
other objects/arrays. Thus implementing the adapter pattern in nearly no time.

##Releases 

### release 1.0
* PHP >= 8.0 (8.0 tested)
### release 0.0
